Musical Scales
=====================================

This is the documentation for the `musical-scales` package,
a lightweight package for giving the notes in a given mode starting on a given note.
A list of modes, examples, and getting started instructions
can be found in the :doc:`README <README>`,
with package information on the :doc:`musical_scales` page.
